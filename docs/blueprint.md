# **App Name**: SiteWise PM

## Core Features:

- User Role Management: Define and manage user roles (Admin, PM, Technician) with associated permissions.
- Site and Task Structure: Define a hierarchical structure for sites, static tasks (common across sites), and dynamic tasks (specific to a site).
- PM Assignment: Assign PMs to sites on a weekly basis. Handle cases where a site might not have a PM assigned.
- Technician PM Completion: Allow technicians to complete PM tasks for a site, including uploading images, adding text descriptions, capturing GPS location automatically, ticking checklist items, and handling task-specific fields.
- Change Request Management: Enable the creation and management of Change Requests (CR) for each site, including descriptions, images, priority, and status tracking (Open, In Progress, Completed, Rejected).
- PM Report Generation: Automatically generate PM reports with photos, descriptions, GPS locations, and checklist status.
- Offline Data Handling and Sync: Enable offline access, data caching, and automatic synchronization when the application is back online.

## Style Guidelines:

- Primary color: Dark cyan (#34A0A4) to inspire feelings of reliability and competence in the users.
- Background color: Light cyan (#E0F4F5).
- Accent color: Steel blue (#468B97) for supplementary information and buttons.
- Body and headline font: 'Inter' sans-serif for a neutral, clean and objective feel.
- Use consistent and clear icons to represent different tasks and statuses.
- A clean, well-organized layout will support smooth navigation and ease of use.
- Subtle transitions and loading animations provide feedback to the user.
